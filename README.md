ShadowIntern 🚀

ShadowSync – Multiplayer Virtual Company Simulation Engine. Students level up in a shared company with real tickets, deadlines, chaos events, and AI feedback.


🎯 What is ShadowIntern?
Students work together in one persistent virtual company:

Shared KPIs: Revenue, churn, uptime, bugs – everyone's work affects them

XP + Levels: Junior → Mid → Senior based on tickets + feedback

Chaos Events: Crunch time, sudden deadlines, team member leaves

AI Features: Real feedback, ShadowAudit perception, Highlight Reels
